# Design Direction

## Design principle

Make it **feel old-school on purpose**, while being modern in:

- spacing and readability
- responsive layout
- image performance
- accessibility

## Layout

- Centered max-width layout
- Desktop: sidebar-style navigation + main content panel
- Generous whitespace (section padding, comfortable line lengths)
- Clear headings and scannable sections (not “wall of text”)

## Typography

Two-font approach:

- Headings: classic serif (heritage feel)
- Body/UI: clean sans (readable, modern)

Guidelines:

- Use strong type hierarchy
- Prefer shorter paragraphs + headings
- Bullets for policies/amenities

## Color and UI

- Neutral “paper/sand” background
- Ink-like text
- One ocean accent for links/CTAs
- Thin borders, subtle rounded corners
- Avoid heavy drop shadows and glossy effects

## Components to standardize early

- Layout (header/nav/footer + meta)
- Hero
- CottageCard
- AmenitiesList
- Gallery (modern photos)
- CTA block (call/email/request dates)
- Archive badge (for legacy photos)

## Tone

- Keep the human voice and signature lines where they matter
- Light tightening is fine (clarity), but avoid “corporate” rewriting
