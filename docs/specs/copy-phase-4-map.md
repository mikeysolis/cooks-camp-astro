# Phase 4 Copy Map

This phase covers the individual cottage markdown files.

## Source Rules

- Use the historic cottage descriptions directly where they are strong and still accurate.
- Keep current-site facts locked:
  - cottage names
  - sleeps
  - bedrooms
  - `half bath`
  - no invented amenities or policies
- Remove all editorial commentary about "historic material," "the current site," or "older copy."

## Direction

- Most cottage pages should read as one strong descriptive paragraph.
- If the historic page already says what the cottage is and what makes it distinct, use that.
- Only adapt when needed for:
  - spelling cleanup
  - `Sandy Scot` naming
  - consistency with locked facts
