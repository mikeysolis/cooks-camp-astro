# Photo Guidelines

## Goals

- New photos look curated and consistent
- Fast loading on mobile
- Legacy photos feel respectful and clearly labeled

## Modern photos (interiors now, exteriors later)

### Export targets
- Hero images: ~1600px wide (or larger if you have very high quality; keep file size reasonable)
- Gallery images: ~1200px wide
- Use JPEG/WebP/AVIF as appropriate (consistent approach per batch)

### Consistency tips
- Keep horizons level in exteriors
- Favor consistent lighting per set
- Avoid near-duplicates (pick best 8–15 per cottage, not 50)

## Legacy photos (archive)

### Treatment
- Never mix into the main modern gallery without labeling
- Prefer a subtle “paper/frame” look over heavy filters
- Use “From the Archive” badges and captions when available

### Scanning and prep
- If scanning: keep originals safe; export web versions separately
- If photos vary wildly in color, do minimal correction for clarity (don’t over-modernize)

## Performance basics

- Don’t ship 8–15MB images
- Prefer fewer high-quality images over huge dumps
- Ensure dimensions are known to prevent layout shift (CLS)

## Accessibility

- Provide alt text for meaningful images when possible
- Decorative images should use empty alt text (`""`)
- If MVP cannot support per-image alt text, use a safe default and plan for an upgrade path
